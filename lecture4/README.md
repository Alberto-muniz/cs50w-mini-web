# Python Console Cash Register

## Description
This project is a simple console-based cash register written in Python.
It was created as a practice project while studying CS50, covering concepts
from Lecture 1 to Lecture 4.

The program simulates a small store register where products can be added,
listed, totaled, and checked out through a menu-driven interface.

## Features
- Add products with name and price
- Show cart contents
- Calculate total price
- Checkout and clear the cart
- Menu-based console interaction

## Concepts Used
- Variables and input/output
- Conditionals and loops
- Functions
- Lists and dictionaries
- List of dictionaries

## Technologies
- Python 3

## Notes
This project is for learning purposes and demonstrates foundational
Python programming skills.
